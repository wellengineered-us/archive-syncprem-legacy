Why SyncPrem?

The number of Software-as-a-Service (SaaS) solutions requiring access to on-premises data is increasing.

Traditional extract, transform, and load (ETL) between on-premises and cloud environments is complex and brittle.

Vendor mandated, purpose-built infrastructure requires large up-front capital investment and on-going operational costs.

Building a specialized on-premises to cloud data movement and mapping platform that readily scales is no trival task.

Inbound firewall rules, installs of data prcoessing daemons, and vendor access to production is a security nightmare.

Users should be delighted with fully syndicated data between their on-premises systems and their cloud solution.

What is SyncPrem?

SyncPrem is a platform that enables data integration and synchronization between on-premises and cloud environments without the complexity and overhead of traditional approaches.

The SyncPrem platform combines custom hardware (appliance) and software components to get on-premise data to the cloud and back – securely, efficiently, and economically. For example, SyncPrem is working with a healthcare cloud SaaS vendor. The vendor drop-ships a low cost, disposable SyncPrem appliance to their customers needing to integrate on-premise EMR/EHR data with the cloud/SaaS product. The vendor's Client Implementation Team works with their customer to build an ideal integration profile for which is stored in the SyncPrem cloud and recognized by the SyncPrem appliance. Data flow can flow in both directions based on thier unique needs! The best part is if for any reason, the appliance begins to exhibit hardware or software issues, these can be swapped out and cloud integration profile is automatically recognized. Additional SyncPrem appliances and be used to scale a tenant horizontally without needing load balancers or complex infrastructure - it simply works!
